# simple_rock_scissors_paper

간단한 웹 기반 가위바위보 게임(rock-paper-scissors)입니다.  
Flask로 구현되어 있고, 향후 카메라로 사용자의 손 모양을 인식하는 기능을 추가할 예정입니다.

## 사용법

1. `pip install flask`로 필요한 패키지 설치
2. `python app.py`로 서버 실행
3. 브라우저에서 [http://localhost:5001](http://localhost:5001) 접속

## 파일 구성

- `app.py`: 메인 서버 코드
- `templates/index.html`: 메인 페이지
